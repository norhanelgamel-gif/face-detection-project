# Face Detection using YOLOv8

This project demonstrates **face detection using YOLOv8 (Ultralytics)**.

## Features
- Train a custom dataset for face detection
- Detect faces in images with bounding boxes
- Display confidence scores for each detection
- Streamlit app for interactive testing

## Dataset
- Dataset used: [Face Detection Dataset on Kaggle](https://www.kaggle.com/datasets/fareselmenshawii/face-detection-dataset)

## Requirements
- Python 3.10+
- Ultralytics YOLO (`pip install ultralytics`)
- Streamlit (`pip install streamlit`)
- OpenCV (`pip install opencv-python`)
- PIL (`pip install pillow`)
